Hashicorp Terraform is an open-source tool for provisioning and managing cloud infrastructure. 
It codifies infrastructure in configuration files that describe the topology of cloud resources. ... 
The Terraform CLI provides a simple mechanism to deploy and version the configuration files to Azure.

Github is a web-based platform used for version control. 
Git simplifies the process of working with other people and makes it easy to collaborate on projects. 
Team members can work on files and easily merge their changes in with the master branch of the project.
